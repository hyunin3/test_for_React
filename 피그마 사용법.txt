## 피그마 사용법

https://niltok.oopy.io/49026060-840e-4075-a19c-727df6af62e4

1. 좌측의 Drafts[+]→Design file을 눌러 새 파일 만들기

2. #모양으로 프레임 선택 가능(다양한 기기 모양 선택 가능)

3. 여러개의 페이지와 여러개의 프레임 사용가능

4. T 버튼 눌러서 텍스트 객체 생성 가능(로그인/회원가입 등등)
텍스트 객체를 선택 후 Shift + A를 누르면 오토 레이아웃이라는 기능이 실행

5. 우측에 보이는 디자인 속성 화면을 통해 
텍스트 객체 간의 간격, 컬러, 백그라운드 설정 등이 가능

6. 좌측 상단의 메뉴에서 Plugins→Browser plugins in Community
이곳에서 다른 유저들의 만들어 둔 디자인 파일이나 리소스, 플러그인 등을 사용 가능
파일 같은 경우에는 Duplicate 후 본인의 파일에 복사해 가져와 사용

7. 우측의 Design, Prototype 탭 옆에 Inspect를 선택하면 해당 객체의 
상세 속성을 볼 수 있음
